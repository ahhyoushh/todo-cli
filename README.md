
# Todo CLI

I create this to get my pthon basics done. Its  a cli-todo List which can be intresting to check out the code.



## Demo

#### Help
##### Note:- On windows replace ```./todo.sh``` with ```todo.bat```.
```http
  ./todo.sh help
```

#### Add item

```http
  ./todo.sh add "Name"
```

####  Delete item
```http
  ./todo.sh del "Number"
```
#### Show remaining items
```http
  ./todo.sh ls
```

#### Complete a todo
```http
  ./todo.sh done `NUMBER`
```

#### Stats
```http
  ./todo.sh report 
```
